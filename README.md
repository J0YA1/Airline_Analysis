# ✈️ Airlines Price Prediction Analysis — Dashboard Assignment

## 📋 Project Overview

This assignment involves building **two interactive data visualization dashboards** — one in **Tableau** and one in **Power BI** — using an Airlines Price Prediction dataset. The goal is to explore how flight prices vary across multiple factors and present findings in a clear, visual, and business-relevant manner.

The project is targeted at helping a **marketing manager** compare and understand flight pricing trends based on real-world variables.

## 🎯 Objectives

Create **2 dashboards** answering the following key business questions:

### Required Analysis Questions

| # | Question |
|---|----------|
| a | Does price vary with Airlines? |
| b | How is the price affected when tickets are bought just 1 or 2 days before departure? |
| c | Does ticket price change based on departure time and arrival time? |
| d | How does the price change with changes in Source and Destination? |
| e | How does ticket price vary between Economy and Business class? |

You are also encouraged to add **additional insights** beyond these five questions.

## 💡 Tips & Suggestions

- **Filter by class:** Create a slipper/filter for Economy vs. Business class to allow side-by-side comparison.
- **Time-based analysis:** Use departure/arrival time bins (Morning, Afternoon, Evening, Night) for cleaner visuals.
- **Days before departure:** A line or bar chart works well to show price spikes for last-minute bookings.
- **Route analysis:** A map or matrix visual can effectively show Source → Destination price variation.
- **Airline comparison:** A grouped bar chart or box plot can highlight pricing differences between carriers.
